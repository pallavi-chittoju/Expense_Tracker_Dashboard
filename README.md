# Expense_Tracker_Dashboard
Expense Tracker Dashboard 💰📊
A modern and interactive Expense Tracker Dashboard built using Python and Streamlit. This application helps users manage their finances by tracking income and expenses, visualizing spending patterns, and monitoring budget performance through insightful charts and reports.
Features
✅ Add, edit, and delete transactions
✅ Categorize expenses and income
✅ Real-time balance calculation
✅ Interactive charts and visualizations
✅ Monthly spending analysis
✅ User-friendly dashboard interface
✅ Data persistence and easy record management
Tech Stack
Python
Streamlit
Pandas
Matplotlib / Plotly
SQLite / CSV Storage
Purpose
This project is designed to simplify personal finance management by providing a clear overview of financial activities and helping users make informed budgeting decisions.
